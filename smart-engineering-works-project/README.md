# Pneumatic Sealing Machine Control System

## Description
Designed and fabricated 30 control boxes for pneumatic sealing machines used in industrial packaging.

## Key Contributions
- AutoCAD enclosure design
- Laser-cut fabrication
- Integration of proximity sensors, solenoid valves, pressure gauge
- Multispan UTR-2033 timer configuration (0.6 sec)

## Applications
Food packaging, industrial sealing systems
